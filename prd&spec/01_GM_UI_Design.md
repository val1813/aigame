# AgentWorld GM 后台 UI 设计规格

> **文档类型**：UI Design Spec  
> **版本**：v1.0 | 2026-04-06  
> **阅读对象**：前端工程师、设计师  
> **配套文档**：PRD v2.0 § 5、TechSpec v2.0 § 3

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **GM 优先** | GM 是创作者，不是技术人员。所有配置项要有友好的中文标签和 Tooltip |
| **所见即所得** | 配置 NPC/任务时，右侧实时预览关卡结构图 |
| **渐进披露** | 高级选项（如 LLM Prompt、作弊锁）默认折叠，避免信息过载 |
| **无障碍保存** | 任何页面刷新/离开前自动存草稿，不丢失 GM 工作 |

## 4. 交互规格

### 4.1 自动保存
- 所有表单字段变更后 **2 秒防抖**触发草稿保存（`PATCH /gm/worlds/{id}/draft`）
- 顶部显示「💾 已自动保存 10:23:45」或「⚠️ 保存失败，请检查网络」

### 4.2 发布流程
```
点击「发布」
    ↓
前端验证（必填项、节点图连通性、胜利条件非空）
    ↓
弹出确认对话框：
「发布后玩家可见，是否确认？
  ⚠️ 发布后修改不影响进行中的 Session」
    ↓
[确认发布] → POST /gm/worlds/{id}/publish
    ↓
关卡加密打包 → CDN 推送（服务器异步，前端轮询状态）
    ↓
状态变为 ✅ 已发布，显示分享链接
```

### 4.3 地图编辑器（Tiled JSON 导入）
- 支持拖拽上传 `.tmj` / `.json` 文件
- 上传后解析并渲染预览（使用 Phaser 的 StaticTilemapLayer，与客户端完全一致）
- 地图最大限制：**100 × 80 Tiles**，超出提示裁切
- 在线编辑：仅支持放置 NPC 图标、标记区域、设置触发点，不支持绘制 Tile（让 GM 用 Tiled 软件绘制后导入）

| 断点 | 行为 |
|------|------|
| ≥1280px | 三栏完整布局 |

## 6. 组件库

使用 **shadcn/ui**（基于 Radix UI + Tailwind），不需要从零实现：

| 用到的组件 | 对应 GM 界面 |
|-----------|-------------|
| `Dialog` | NPC 编辑抽屉、发布确认 |
| `Tabs` | 关卡编辑器左侧面板切换 |
| `Select` / `Combobox` | 难度、触发条件选择 |
| `Textarea` | Boss Prompt 编辑、背景故事 |
| `Badge` | 状态标签（草稿/已发/已归档）|
| `Toast` | 保存成功/失败提示 |
| `Table` | 关卡列表、审核队列 |
| `Progress` | 发布进度 |

节点图编辑器：使用 **React Flow**（开源，无需联网安装后离线可用）

```bash
npm install reactflow
```

示例（任务节点图最小实现）：

```tsx
import ReactFlow, { addEdge, useNodesState, useEdgesState } from 'reactflow';
import 'reactflow/dist/style.css';

const initialNodes = [
  { id: 'start',   position: { x: 0,   y: 0   }, data: { label: '开始' } },
  { id: 'node_a',  position: { x: 200, y: 0   }, data: { label: '接触张三' } },
  { id: 'node_b',  position: { x: 400, y: 0   }, data: { label: '获取地址' } },
];
const initialEdges = [
  { id: 'e1', source: 'start',  target: 'node_a', animated: true },
  { id: 'e2', source: 'node_a', target: 'node_b', animated: true },
];

export function QuestGraph() {
  const [nodes, , onNodesChange] = useNodesState(initialNodes);
  const [edges, setEdges, onEdgesChange] = useEdgesState(initialEdges);
  return (
    <div style={{ height: 400, background: '#1E293B', borderRadius: 8 }}>
      <ReactFlow
        nodes={nodes} edges={edges}
        onNodesChange={onNodesChange}
        onEdgesChange={onEdgesChange}
        onConnect={(p) => setEdges((eds) => addEdge(p, eds))}
        fitView
      />
    </div>
  );
}
```
